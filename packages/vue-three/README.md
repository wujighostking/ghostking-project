# tsdown-starter

A starter for creating a TypeScript package.

## Development

## Editor

This template ships with a `.editorconfig` file.

- UTF-8 encoding
- LF line endings
- 2-space indentation
- final newline on save

Keep it together with your formatter configuration so editor behavior stays consistent across environments.

- Install dependencies:

```bash
npm install
```

- Build the library:

```bash
npm run build
```
